# MediaCompressorPro --- Configuração do Ambiente de Desenvolvimento

## Requisitos

-   Python 3.10 ou superior
-   Git
-   Visual Studio Code

## 1. Clonar o repositório

``` bash
git clone https://github.com/mtutida/mediacompressor.git D:\apps\mediacompressor
ou se já estiver na pasta pai:
git clone https://github.com/mtutida/mediacompressor.git mediacompressor
```

## 2. Entrar na pasta do projeto

``` bash
cd mediacompressor
```

## 3. Criar ambiente virtual

``` bash
python -m venv .venv
```

## 4. Ativar o ambiente

Windows:

``` bash
.venv\Scripts\activate
```

## 5. Instalar dependências

``` bash
pip install -r requirements.txt
```

## Resultado esperado

O projeto estará pronto para execução local.
