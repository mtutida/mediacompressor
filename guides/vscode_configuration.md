# MediaCompressorPro --- Configuração do VS Code

## Selecionar interpretador Python

1.  Pressione:

```{=html}
<!-- -->
```
    Ctrl + Shift + P

2.  Digite:

```{=html}
<!-- -->
```
    Python: Select Interpreter

3.  Escolha:

```{=html}
<!-- -->
```
    .venv\\Scripts\\python.exe

## Verificar interpretador ativo

No terminal:

    where python

O resultado deve apontar para o `.venv` do projeto.
