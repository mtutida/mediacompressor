# MediaCompressorPro

Ferramenta desktop para **compressão eficiente de mídia** (vídeo, áudio
e imagens) com foco em:

-   processamento em lote
-   interface visual simples
-   automação de compressão
-   redução significativa de tamanho de arquivos

O projeto é desenvolvido em **Python** com interface gráfica e
arquitetura modular.

------------------------------------------------------------------------

# Principais recursos

-   Compressão de múltiplos arquivos
-   Interface gráfica intuitiva
-   Pipeline de processamento de mídia
-   Arquitetura modular para expansão
-   Preparado para processamento em lote

------------------------------------------------------------------------

# Instalação rápida

Clone o repositório:

    git clone https://github.com/mtutida/mediacompressor.git D:/apps/mediacompressor

Entre na pasta:

    cd D:/apps/mediacompressor

Crie o ambiente virtual:

    python -m venv .venv

Ative o ambiente:

    .venv\Scripts\activate

Instale dependências:

    pip install -r requirements.txt

------------------------------------------------------------------------

# Executar aplicação

Via terminal:

    cd src
    python -m app

Ou via VS Code:

    Ctrl + Shift + B
